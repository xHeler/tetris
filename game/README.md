# Tetris game
Tetris' game written using pygame.

# Setup
Clone repository.
```shell
git clone https://github.com/xHeler/tetris.git
cd game
```
Create environment and install dependencies.
```shell
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
Run a game.
```shell
python src.main
```
